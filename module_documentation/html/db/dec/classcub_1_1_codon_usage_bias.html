  seq,
  genetic_code = 1

Method to generate codon counts from sequence.

This method will take the nucleotide sequence to translate into amino acid sequence before codon count generation for each amino acid, resulting in codon counts per amino acid.

Parameters
seqString: Nucleotide sequence string
genetic_codeInteger: Genetic code number to be used for translation. Default = 1 (Standard Code). For more information, see https://www.ncbi.nlm.nih.gov/Taxonomy/Utils/wprintgc.cgi
Returns
A nested dictionary of {'amino acid': {'codon': count}}
