16 and had started to translate some of them into Python during some overly
17 energetic days. Finally, Python Cookbook had undergone 2 editions by 2008 and
18 ActiveState had provided an online platform for Python Recipes which I found to
19 be useful and can see how some of these recipes can be merged. Thus, COPADS is
