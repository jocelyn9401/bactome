27 def test(statistic, distribution, confidence):
28  """Generates the critical value from distribution and confidence value
29  using the distribution's inverseCDF method and performs 1-tailed and
30  2-tailed test by comparing the calculated statistic with the critical
31  value.
32 
33 
34  Returns a 5-element list
35  [left result, left critical, statistic, right critical, right result]
36  where
37  - left result = True (statistic in lower critical region) or
38  False (statistic not in lower critical region)
39  - left critical = lower critical value generated from 1 - confidence
40  - statistic = calculated statistic value
41  - right critical = upper critical value generated from confidence
42  - right result = True (statistic in upper critical region) or
43  False (statistic not in upper critical region)
44 
45  Therefore, null hypothesis is accepted if left result and right result are
46  both False in a 2-tailed test.
47 
48  @param statistic: calculated statistic (float)
49  @param distribution: distribution to calculate critical value
50  @type distribution: instance of a statistics distribution
51  @param confidence: confidence level of a one-tail
52  test (usually 0.95 or 0.99), use 0.975 or 0.995 for 2-tail test
53  @type confidence: float of less than 1.0"""
54  data = [None, None, statistic, None, None]
55  data[1] = distribution.inverseCDF(1.0 - confidence)[0]
56  if data[1] < statistic: data[0] = False
57  else: data[0] = True
58  data[3] = distribution.inverseCDF(confidence)[0]
59  if statistic < data[3]: data[4] = False
60  else: data[4] = True
61  return data
